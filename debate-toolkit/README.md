# Debate Toolkit

An APDA parliamentary debate plugin for Claude. Includes two skills for competitive debaters:

## Skills

### Case Prep
Generate full debate cases with framework, contentions, evidence, and preemptive responses. Works for both Government and Opposition sides on any motion.

Trigger with: "prep a case", "build a case for [motion]", "gov case for...", "opp case for..."

### Rebuttal Generator
Analyze opponent arguments and produce targeted rebuttals with logical, empirical, and impact-level attacks. Includes ready-to-deliver spoken lines.

Trigger with: "rebut this argument", "counter this point", "respond to their case"

## Setup

No external services or API keys required. Install the plugin and the skills are available immediately.

## Usage

Provide a debate motion and specify Gov or Opp to get a full case. Paste in an opponent's arguments to get structured rebuttals. Both skills are tailored to APDA parliamentary format with speech times, POI rules, and strategic considerations built in.
